API Documentation

POST /signup

Registers a new user.

Request Body:

{
    "Email": "user@example.com",
    "Password": "password123"
}

Response:

{
    "message": "User created successfully!",
    "Email": "user@example.com",
    "Password": "password123"
}

POST /login

Logs in an existing user.

Request Body:

{
    "Email": "user@example.com",
    "Password": "password123"
}

Response:

{
    "success": true,
    "message": "Login successful"
}

POST /add

Adds new data entry with a title.

Request Body:

{
    "Title": "My new entry"
}

Response:

{
    "success": true,
    "data": {
        "ID": 1,
        "CreatedAt": "2024-09-30T12:00:00Z",
        "UpdatedAt": "2024-09-30T12:00:00Z",
        "DeletedAt": null,
        "Title": "My new entry",
        "Body": []
    }
}

GET /add

Retrieves all data entries.

Response:

{
    "data": [
        {
            "ID": 1,
            "Title": "My new entry",
            "Body": []
        }
    ]
}

GET /data/<dataName>/body

Retrieves the body of a specific data entry by title.

Path Parameters:

dataName: The title of the data entry

Response:

[
    "Item 1",
    "Item 2"
]

POST /add/<dataName>/body

Adds an item to the body of a specific data entry by title.

Path Parameters:

dataName: The title of the data entry

Request Body:

{
    "Body": "New body item"
}

Response:

{
    "success": true,
    "Body": [
        "Item 1",
        "Item 2",
        "New body item"
    ]
}

PUT /data/<dataName>/body/<index>

Edits a specific item in the body of a data entry by index.

Path Parameters:

dataName: The title of the data entry
index: The index of the item to edit

Request Body:

{
    "Body": "Updated body item"
}

Response:

{
    "success": true,
    "Body": [
        "Item 1",
        "Updated body item"
    ]
}

DELETE /data/<dataName>/body/<index>

Deletes a specific item in the body of a data entry by index.

Path Parameters:

dataName: The title of the data entry
index: The index of the item to delete

Response:

{
    "success": true,
    "Body": [
        "Item 1"
    ]
}
